# 🎵 Se Liga na Música (Music Preferences Analysis)

Projeto de análise exploratória de dados utilizando Python e Pandas para identificar diferenças nas preferências musicais de usuários em diferentes cidades.

## Objetivo

Analisar os dados de streaming musical para comparar os hábitos de escuta entre duas cidades e identificar padrões de comportamento.

## Tecnologias utilizadas

- Python
- Pandas
- Jupyter Notebook

## Estrutura do projeto

## Principais análises

- Limpeza dos dados
- Tratamento de valores ausentes
- Remoção de duplicatas
- Comparação entre cidades
- Comparação de preferências musicais por dia da semana
