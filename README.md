# 🎵 Music Preferences Analysis

Projeto de Análise Exploratória de Dados desenvolvido em **Python** utilizando **Pandas** para comparar os hábitos de consumo musical entre usuários de diferentes cidades.

---

# 📌 Objetivo

Analisar dados de streaming musical para identificar diferenças nas preferências musicais entre duas cidades, verificando padrões de comportamento dos usuários ao longo da semana.

---

# 📖 Descrição do Projeto

Este projeto utiliza uma base de dados de streaming musical para realizar uma análise exploratória, comparando os hábitos de escuta de usuários em diferentes cidades.

Foram aplicadas técnicas de limpeza, tratamento e análise dos dados para validar hipóteses sobre o comportamento dos usuários e identificar possíveis diferenças nas preferências musicais.

---

# 🔎 Metodologia

O projeto foi desenvolvido seguindo as seguintes etapas:

- Importação dos dados;
- Limpeza e preparação da base;
- Tratamento de valores ausentes;
- Remoção de registros duplicados;
- Padronização das informações;
- Comparação entre as cidades;
- Comparação das preferências musicais por dia da semana;
- Validação das hipóteses propostas.

---

# 📊 Resultados

A análise permitiu:

- Comparar os hábitos de escuta entre as cidades analisadas;
- Identificar diferenças nas preferências musicais;
- Verificar padrões de comportamento ao longo da semana;
- Confirmar ou refutar as hipóteses propostas com base nos dados.

# ▶️ Como executar

Clone o repositório:

```bash

```

Entre na pasta:

```bash
cd music-preferences-analysis
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

